CSS Implementation Methods

Inline CSS

Styles applied directly to HTML elements

<p style="color: blue;"> Blue text </p>

Use: Quick fixes, testing

Internal CSS

Styles defined in the <head> section

<style> p { color: blue; } </style>

Use: Single page styling

External CSS

Separate .css file linked to HTML

<link rel="stylesheet" href="styles.css">

Use: Professional projects

CSS Selectors: Target Your Elements

Element Selectors

p { color: red; } h1 { font-size: 2rem; }

Targets all elements of a specific type

Class Selectors

.highlight { background: yellow; } .btn { padding: 10px; }

Targets elements with specific class

ID Selectors

#header { background: blue; } #main-content { ... }

Targets unique elements with specific ID

The CSS Box Model

Every element is a box with four areas:

Content

The actual content (text, images)

Padding

Space between content and border

Border

The border around the padding

Margin

Space outside the border

.box { width: 300px; padding: 20px; border: 10px solid blue; margin: 15px; }

Create flexible, responsive layouts with ease!

Key Properties: justify-content, align-items, flex-direction, flex-wrap

Responsive Design with Media Queries

Make your website look great on all devices!

📱 Mobile First

Design for mobile, then enhance for larger screens

🖥️ Breakpoints

Define different styles for different screen sizes

/* Mobile styles (default) */ .container { width: 100%; } /* Tablet and up */ @media (min-width: 768px) { .container { width: 750px; } } /* Desktop and up */ @media (min-width: 1024px) { .container { width: 1000px; } }
